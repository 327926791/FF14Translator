## 💡 核心改进详解

### 1. 翻译后端扩展

**之前**：只支持 Ollama，需要本地 GPU

**现在**：
- 支持 4 种方案
- 可在线下切换
- 适应不同场景

```python
# 新增翻译器工厂
translator = TranslatorFactory.create(config)
```

### 2. 配置管理系统

**之前**：配置存在 UI 中，每次启动重新设置

**现在**：
- 配置自动保存
- 跨启动持久化
- 支持多配置切换

```python
config = AppConfig.load_from_file()  # 自动加载
config.save_to_file()                 # 自动保存
```

### 3. 改进的 UI

**之前**：只支持 Ollama 配置

**现在**：
- 翻译后端下拉选择
- 选项卡式配置
- 连接检查按钮
- API 快速链接
